# EA Factory Admin QR Machine

ชุดนี้เพิ่มเมนู QR รายเครื่องในหน้า Admin Panel

ไฟล์ที่แก้:
- admin-panel.html
- admin-panel.css
- admin-panel.js

วิธีวางไฟล์:
- admin-panel.html -> วางแทนหน้า Admin เดิม
- admin-panel.css -> /css/admin-panel.css
- admin-panel.js -> /js/admin/admin-panel.js

สิ่งที่เพิ่ม:
1. เมนู ⚙️ QR รายเครื่อง
2. เลือกแผนกแล้วแสดงเครื่องจักรของแผนกนั้น
3. คัดลอกลิงก์รายเครื่อง
4. เปิด QR รายเครื่อง
5. พิมพ์ QR ทั้งแผนก
6. ปุ่ม QR ในรายการเครื่องจักรของหน้า Master Data
